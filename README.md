#API

- NodeJS
- Web Framework - Fastify
- Database - MongoDB
- Authentication - JWT

#Requerimentos

- node > 8.XX
- npm > 6.XX

#Instalação

```bash
npm install

'Caso o Mongo não esteja rodando na porta 27017 substitua pela porta correta no .env'

npm run dev
```
